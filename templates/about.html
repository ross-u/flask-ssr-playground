About

What is this project about?

The project is built using a tech stack that includes SQLite database, Python and Flask for the backend. Additional Flask extensions like Flask-Mail and Flask-Session are used for email handling and session management. I used Mailtrap's email service, mainly for the simplicity of the setup, and it is used for sending account verification emails. To make the prototyping and styling faster, Bootstrap is used for the frontend.
